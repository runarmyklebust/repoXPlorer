# repoXPlorer

The repoXPlorer is a tool for querying and managing Enonic XP repositories.
_Bootstrap4 seems to have som issues in Firefox, so Chrome is recommended._

| ----------- | ------------------- 
| 1.0.0 | 6.9.0 | 
| 2.0.1 | 7.0.0 |
